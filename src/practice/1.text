export class 1Class{}와
export default class 2Class{}의 차이점

1Class
이름이 있는 named 내보내기
이 방식은 이름이 고정됩니다.
가져올 때 반드시 해당 이름으로 가져와야 합니다.

2Class
기본 내보내기
모듈에서 단 하나의 기본 값만 내보낼 때 사용됩니다.
클래스 이름은 파일 내부에서는 사용되지만, 가져올 때 원하는 이름으로 변경할 수 있습니다.

장점: 가져올 때 이름을 자유롭게 정할 수 있습니다.
주로 모듈에서 가장 중요한 값이나 클래스를 대표하는 경우 사용합니다.

한 파일당 한번만 사용할 수 있습니다.
이름이 명시적이지 않아 협업시 헷갈릴수있습니다.

모듈에서 여러 기능으로 내보낼 때는 export class처럼 이름 있는 내보내기를 주로 사용하고,
모듈에서 하나의 대표적인 값이나 클래스를 내보낼 때는 export default 를 사용하는 것이 일반적입니다.
